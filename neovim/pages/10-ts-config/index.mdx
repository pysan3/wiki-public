# Setup Treesitter for Syntax Highlights

## What is Treesitter

[Tree-sitter](https://tree-sitter.github.io/tree-sitter/) はパーサー生成ツールであり、インクリメンタルな構文解析ライブラリです。ソースファイルに対して具体的な構文木を構築し、ソースファイルの編集に伴って構文木を効率的に更新することができます。GitHub のウェブサイトでコードに色をつけているのはこれを使っています。

これを Nvim で使えるようにしたのが [nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter) で、これをインストールしましょう。

左：なし、右：あり
![nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter/raw/master/assets/example-cpp.png)

### Requirements

- `nvim >= 0.7.0`
- `tar`, `curl`, `git`
- C compiler and `libstdc++`

## Plugin Install

```lua filename="10-ts-config/init.lua"
return {
  setup = {
    "folke/todo-comments.nvim",
    "andymass/vim-matchup",
    {
      "nvim-treesitter/nvim-treesitter",
      run = ":TSUpdate",
      requires = {
        "nvim-treesitter/nvim-treesitter-textobjects",
        "andymass/vim-matchup",
      },
    },
    "m-demare/hlargs.nvim",
  },
}
```
