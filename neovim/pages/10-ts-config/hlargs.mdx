# hlargs.nvim: [GitHub](https://github.com/m-demare/hlargs.nvim)

デフォルトの Treesitter では関数の引数と関数内変数の色は同じになるが、関数の引数だけ別の色で塗り分けてくれるプラグイン。

## Setup

ハイライトの色を設定で決めることができる。デフォルトは上の画像のようなオレンジ色だが、自分の好みの colorscheme に合わせて変更してください。

```lua filename="10-ts-config/hlargs.lua"
require("hlargs").setup({
  -- color = "#89d0ff",
})
```
